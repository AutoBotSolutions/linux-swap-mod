# Linux Swap Optimizer Website

This directory contains the state-of-the-art sci-fi themed website for the Linux Swap Optimizer project, designed for GitHub Pages deployment.

## Features

- **Sci-Fi Theme**: Modern, futuristic design with neon colors and holographic effects
- **Responsive Design**: Fully responsive layout for all device sizes
- **Interactive Elements**: Smooth animations, hover effects, and dynamic content
- **Multiple Pages**: Home, Features, Documentation, and Installation pages
- **Performance Optimized**: Fast loading with minimal dependencies

## Files

- `index.html` - Main landing page
- `features.html` - Detailed features page
- `docs.html` - Documentation hub
- `install.html` - Installation guide
- `styles.css` - Complete styling with sci-fi aesthetics
- `script.js` - Interactive JavaScript functionality
- `.nojekyll` - GitHub Pages configuration file

## Deployment

### GitHub Pages

1. Push this repository to GitHub
2. Enable GitHub Pages in repository settings
3. Select `docs/` folder as source
4. The site will be available at `https://username.github.io/linux-swap-mod/`

### Local Development

Simply open `index.html` in a web browser. No build process required.

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #00f0ff;
    --secondary-color: #7b2cbf;
    --accent-color: #ff006e;
    --bg-dark: #0a0a0f;
    --bg-darker: #050508;
}
```

### Content
Edit HTML files to customize text, links, and content sections.
