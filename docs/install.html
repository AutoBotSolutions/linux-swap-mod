INSTALLATION PROCESS

Follow these steps to deploy the system

01

CLONE REPOSITORY

Clone the repository to your local system

git clone https://github.com/AutoBotSolutions/linux-swap-mod.git
02

NAVIGATE TO DIRECTORY

Change to the project directory

cd linux-swap-mod
03

RUN INSTALLATION SCRIPT

Execute the installation script with sudo privileges

sudo bash install.sh
04

ENABLE SERVICE

Enable the systemd service to start on boot

sudo systemctl enable swap-optimizer
05

START SERVICE

Start the optimization service immediately

sudo systemctl start swap-optimizer
06

VERIFY INSTALLATION

Check the service status to ensure it's running

sudo systemctl status swap-optimizer

CONFIGURATION

Customize the system for your needs

The configuration file is located at /etc/swap-optimizer/config.json. You can edit this file to customize optimization parameters.

sudo nano /etc/swap-optimizer/config.json
VIEW CONFIGURATION GUIDE →

MANUAL INSTALLATION

Alternative installation method

Install Dependencies

pip3 install -r requirements.txt

Copy Files

sudo cp swap_optimizer.py /opt/swap-optimizer/ sudo cp config.json /etc/swap-optimizer/

Install Service

sudo cp swap-optimizer.service /etc/systemd/system/ sudo systemctl daemon-reload

TROUBLESHOOTING

Common issues and solutions

Service Won't Start

Check logs with: sudo journalctl -u swap-optimizer -f

Permission Denied

Ensure you're running with sudo or as root user

Python Not Found

Install Python 3: sudo apt install python3

Dependencies Missing

Install pip: sudo apt install python3-pip
